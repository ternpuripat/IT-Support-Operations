# Government IT Support & Operations

## Overview
Provided IT support and operational assistance within a government organization, ensuring smooth daily operations for staff and internal systems.

## Responsibilities

### IT Support
- Provided technical support for staff, resolving issues related to Windows OS, software, and hardware
- Diagnosed and troubleshooted system, network, and device-related problems
- Installed, configured, and maintained computers and office equipment

### Application Support
- Assisted users with internal applications and software usage
- Resolved system errors and configuration issues
- Provided user guidance and basic training

### Hardware & Infrastructure
- Maintained printers, computers, and office IT equipment
- Diagnosed hardware failures and performed basic repairs or replacements

### Operations & Coordination
- Supported administrative tasks including procurement and coordination
- Assisted in managing digital content and public communication materials

### Design & Communication
- Created basic visual content using Canva, Figma, and Adobe tools
- Supported public relations and internal communication tasks

## Skills Demonstrated
- Troubleshooting & problem-solving
- User support & communication
- System setup and maintenance
- Multi-tasking in operational environment
